Speak11

Select text in any app, press ⌥⇧/, hear it read aloud.
Cloud TTS via ElevenLabs, or local TTS via Kokoro. Runs in your menu bar.

Download View on GitHub

Requires macOS 13+. ElevenLabs needs an API key. Local TTS needs Apple Silicon.

One hotkey, any app

Works in browsers, editors, Electron apps, anything. The hotkey is system-level, so no app can block it.

Pick a voice

ElevenLabs voices or 12 Kokoro voices built in. Paste any ElevenLabs voice ID for more.

Tune it

Speed, stability, similarity, style, all adjustable from the menu bar. Changes apply instantly.

No dependencies

Uses curl and afplay, which already ship with macOS. Double-click the installer and you're set.

Works offline

Local TTS via Kokoro runs entirely on your Mac. No internet, no credits, no limits. Apple Silicon only.

Choose your engine

ElevenLabs for cloud quality, Kokoro for offline freedom. Install both and switch from the menu bar.

Key stays in Keychain

Your ElevenLabs API key goes straight into the macOS Keychain. It's never written to a file.

Free and open

Released under the Unlicense. Do whatever you want with it.
